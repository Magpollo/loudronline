## Getting Started

### Prerequisites

- Nodejs v20.0.0
- npm v7.0.0

### Installation

1. Clone the repo

```sh
git clone
```

2. Install NPM packages

```sh
npm install
```

3. To run the frontend

```sh
cd frontend
npm run dev
```

4. To run the backend

```sh
cd backend
npm run develop
```
